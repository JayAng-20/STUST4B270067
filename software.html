ROK Control Center

把鍵盤設定變成一套可保存的工作流程

燈效、巨集、按鍵重映射、磁軸觸發點與隨機延遲曲線，都能在 ROK Control Center 中建立設定檔。

設定檔可以跟著場景走

上班時使用安靜燈效與媒體鍵，遊戲時切換高回報率與 Rapid Trigger，剪輯時把巨集交給旋鈕與 Fn 層。

  • 每鍵重映射與多層 Fn 設定。
  • RGB 分區、音訊視覺化與分享碼匯入。
  • 磁軸機種支援 0.1mm 級觸發調整。
  • 可控隨機延遲，讓巨集更接近自然輸入節奏。

Modules

核心功能

Keymap Studio

拖曳設定按鍵、建立 Fn 層，並把常用快捷鍵保存到鍵盤記憶體。

RGB Composer

用色帶、節奏與分區做出商務低調或電競高亮的燈效。

Trigger Lab

磁軸機種可調觸發點、快速重置、死區與遊戲專用設定檔。

Windows 10+

完整支援燈效、巨集、韌體更新。

macOS 13+

支援按鍵重映射、設定檔與韌體更新。

Web Config

部分機種可透過瀏覽器快速調整鍵位。
